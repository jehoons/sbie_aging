# Usr directory 
Packages are installed in the usr directory. 

## Packages 

### MaBoSS - boolean network simulator

for linux os

```
wget https://maboss.curie.fr/pub/MaBoSS-env-2.0.tgz
cd MaBoSS-env-2.0/
./check-requirements.sh 
cd engine/src
make 
# or 
make install
# Then, you can find the bin file MaBoSS.  
```
